# Design System

A Vue 3 design system built with TypeScript, Tailwind CSS v4, and Storybook.

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Run Storybook:

```bash
npm run storybook
```

### Build

Build the project:

```bash
npm run build
```

Build Storybook:

```bash
npm run build-storybook
```

## Components

- **Button** - A versatile button component with multiple variants and sizes

## Tech Stack

- Vue 3
- TypeScript
- Tailwind CSS v4
- Storybook
- Vite
